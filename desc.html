Course Description

These are all the coursecode for all the courses I'm taking, as well as a brief description of all of them.



Course Code Description
CSC574 Dynamic Web Application
This course focuses on building interactive and dynamic web applications using a range of technologies such as HTML, CSS, JavaScript, and server-side programming languages. You'll learn how to create responsive, scalable, and secure web applications that can run on different devices and platforms.
CSC584 Enterprise Programming
This course covers the principles and practices of programming in enterprise-level systems. You'll learn about topics such as software architecture, design patterns, and enterprise-level frameworks and technologies. You'll also gain hands-on experience in developing enterprise applications using popular programming languages and tools.
CSC577 Software Engineering - Theories and Principles
This course introduces you to the fundamental principles and theories of software engineering, including software development life cycle, requirements analysis, design patterns, and testing. You'll also learn about the various software development methodologies and how to apply them to real-world software development projects.
CSC508 Data Structures
This course covers the fundamental data structures and algorithms used in computer science, including linked lists, stacks, queues, trees, and graphs. You'll learn how to implement these data structures in popular programming languages and use them to solve computational problems.
CSC569 Principles of Compilers
This course teaches you about the principles and techniques used in designing and implementing compilers for programming languages. You'll learn how to analyze and transform source code into executable code, and gain hands-on experience in developing a simple compiler.
TMC451 Introductory Mandarin
This course introduces you to the basic Mandarin Chinese language skills, including speaking, listening, reading, and writing. You'll learn about Chinese culture and customs, as well as useful phrases and vocabulary for everyday communication.
ElC650 English For Professional Interaction
This course focuses on developing your English language skills for professional settings. You'll learn how to communicate effectively in different professional contexts, such as job interviews, business meetings, and presentations. You'll also learn about cultural differences and how to adapt your communication style accordingly.
